# Contribution Guidelines

This document provides basic guidelines for contributors to ensure
a smooth and respectful collaboration experience.

## General Rules
- Be respectful and professional
- Keep changes small and focused
- Follow the instructions mentioned in the issue
- Do not modify unrelated files

## Beginner Contributions
This repository is designed to support first-time contributors.

If this is your first pull request:
- Read the README carefully
- Follow the steps exactly
- Ask questions if something is unclear

## Commit Messages
- Use clear and simple commit messages
- Avoid unnecessary words
- Example: `Add username to contributors list`

## Pull Request Review
All pull requests will be reviewed.
Valid and relevant contributions will be merged.

Thank you for contributing.
