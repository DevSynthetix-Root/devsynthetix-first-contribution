# Contributing Guide

Thank you for your interest in contributing.

This repository is beginner-friendly and focuses on
small, clear contributions.

## Contribution rules
- Make only the changes requested in the issue
- Keep commits clean and relevant
- Follow the repository structure
- Be respectful in discussions

## Pull Request checklist
- [ ] Issue is assigned to you
- [ ] Changes are small and focused
- [ ] Description clearly explains what was done

Maintainers will review and merge valid pull requests.
